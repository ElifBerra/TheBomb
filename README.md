# TheBomb

Oynanış Özeti:
Oyuncu tarafından kontrol edilen bir karakterin/objenin, sadece yatay eksende hareket ederek yukarıdan rastgele düşen nesneleri yakaladığı (veya kaçındığı) temel bir oyun döngüsü tasarlamanızı istiyoruz.

Teknik Gereksinimler:
Projenin çalışır durumda olmasının yanı sıra, kod mimarisinde aşağıdaki yapıların kullanılması zorunludur:
1. Veri Yönetimi (ScriptableObject):
Düşen nesnelerin özellikleri (hız, puan değeri, görsel vb.) kod içine gömülü (hardcoded) olmamalı; ScriptableObject yapısı üzerinden dinamik olarak çekilmelidir.

2. İletişim (Event System):
Oyun mantığı ve Arayüz (UI) arasındaki veri akışı Event (Olay) tabanlı olmalıdır. Sınıflar mümkün olduğunca birbirine doğrudan referans vermemeli, "Loose Coupling" (Gevşek Bağlılık) prensibine uyulmalıdır.

3. Kod Standartları:
Okunabilir, geliştirilmeye açık ve OOP (Nesne Yönelimli Programlama) prensiplerine uygun bir yapı beklenmektedir.

Taskı üzerine yapılmıştır.
